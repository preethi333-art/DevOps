# ✅ Python Files, Running Code & PyCharm — Cheat Sheet

***

## 1. Interactive Console vs Python Files

### Interactive Console (Shell)

*   Executes **one line at a time**
*   Gives **instant output**
*   Good for **practice and testing**

Example:

```python
print("Hello")
```

✅ Output appears immediately

***

### Python File (`.py`)

*   Used for **multiple lines of code**
*   Required for **projects**
*   Code is saved and executed as a program

✅ Python files always end with **`.py`**

***

## 2. Creating a Python File

### Method 1: Using an Editor / IDE

1.  File → **New File**
2.  Write code
3.  Save as:
        filename.py

Example:

```python
print("Hello")
```

***

### Method 2: Using Notepad

1.  Open Notepad
2.  Write Python code
3.  Save as:
        my_code.py
    ✅ Important: Extension must be `.py`

***

## 3. Running a Python File

### Option 1: Run from IDE (IDLE / Editor)

*   Save the file (`Ctrl + S`)
*   Click **Run → Run Module**
*   Or press **F5**

✅ Output appears in the Shell

***

### Option 2: Run from Command Prompt (CLI)

#### Step 1: Navigate to File Location

```bash
cd Desktop
cd Python_Codes
```

#### Step 2: Run the File

```bash
python my_code.py
```

✅ Output appears in the command prompt

***

## 4. Writing a Multi‑Line (Big) Code Example

### Add Two Numbers

```python
x = 5
y = 6
z = x + y
print(z)
```

✅ Output:

    11

***

## 5. Why Use an IDE?

✅ Easier coding  
✅ Syntax highlighting  
✅ Error detection  
✅ Debugging support

Popular IDEs:

*   **PyCharm**
*   IDLE
*   Eclipse
*   VS Code

***

## 6. PyCharm Basics

### Creating a New Project

1.  Open **PyCharm**
2.  Click **Create New Project**
3.  Enter project name
4.  Click **Create**

✅ PyCharm sets up environment automatically

***

### Creating a Python File in PyCharm

1.  Right‑click project
2.  New → **Python File**
3.  Give file name
4.  `.py` extension is automatic

***

### Running Code in PyCharm

*   Click **Run ▶**
*   OR Right‑click file → **Run**
*   OR Use green run button

✅ Output appears in run console

***

## 7. Changing Font Size in PyCharm

1.  File → **Settings**
2.  Editor → **Font**
3.  Change size (e.g., 25)
4.  Click **OK**

✅ Better visibility while coding

***

## 8. Debugging in PyCharm (Very Important)

### What is Debugging?

*   Step‑by‑step execution
*   Watch variable values change
*   Helps find bugs in large projects

***

### Adding a Breakpoint

*   Click next to the **line number**
*   A red dot appears ✅

***

### Starting Debug Mode

1.  Run → **Debug**
2.  Program stops at breakpoint

***

### Stepping Through Code

*   Press **F8** → move to next line

Example Code:

```python
x = 5
y = 8
z = x + y
print(z)
```

✅ Debug view shows:

*   x = 5
*   y = 8
*   z = 13

***

## 9. Why Debugging is Powerful

*   Finds errors in **large files (1000+ lines)**
*   Tracks incorrect values
*   Saves hours of guessing

✅ Essential for professional developers

***

## 10. Key Takeaways

*   `.py` → Python file
*   Use IDE for real projects
*   Save before running
*   Command Prompt & IDE both work
*   Debugging helps locate bugs
*   Learn step by step—don’t rush projects
